LearnLogix

LearnLogix is a learning management system (LMS) designed to manage course registration, learning progress, and certifications for universities or organizations. Built using Node.js, MySQL, and modern web technologies, it allows instructors to create courses, manage enrolled students, upload course materials, and update in-course marks. Students can browse and enroll in courses, track progress, and access learning resources through a personalized dashboard. The system also includes admin functionalities to oversee users, courses, and approvals.

JourneyX

JourneyX is a full-stack web-based bus ticket booking system developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. It allows users to search bus routes, view schedules, select seats, and book tickets through a secure login system. The platform also includes an admin panel to manage buses, routes, schedules, and bookings. JourneyX demonstrates efficient data handling, real-time seat management, and practical implementation of modern web development technologies.
